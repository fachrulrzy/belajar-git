= GIT - Github

= Contoh version control system : git, subversion, mercurial, cvs

= Git
  - Sebuah vcs untuk mengelola perubahan file di dalam folder (repository)
  - git akan menyimpan perubahan file dengan serangkaian "commit"
  - "branch", cabang dalam commit
  - "merge", gabungin branch ke commit utama

= Github
  - git tapi di cloud
  - "pull", taro commit ke pc dari Github
  - "push", ambil commit dari github ke pc

= Glosarium
  - repo : folder projek
  - commit : rekaman perubahan dari repo
  - hash : penanda unik pada sebuah commit
  - checkout : berpindah ke sebuah commit
  - branch : cabang bebas dari sebuah commit
  - merge : gabungin branch
  - remote : sumber yang memiliki repo
  - clone :  mengambil repo dari remote
  - pull : taro commit ke pc dari Github
  - push : ambil commit dari github ke pc
